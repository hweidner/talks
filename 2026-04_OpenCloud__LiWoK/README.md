# VPNs mit Wireguard

_Kurzvortrag im Linux-Workshop Köln am 7. April 2026_

[OpenCloud](https://opencloud.eu/) ist eine webbasierte Software zur Dateien-
Verwaltung und Zusammenarbeit an gemeinsamen Dateien. Es wurde als Fork von
ownCloud Infinite Scale gestartet, was eine Neuimplementierung der Produkte
ownCloud und Nextcloud ist.

Der Vortrag zeigt die Geschichte von OpenCloud, die verschiedenen 
Installationsvarianten und gibt einen Ausblick auf Erweiterungen und
zukünftige Entwicklungen. In einer Demo werden eine Minimalinstallation
sowie die Standardinstallation mit Collabora Online und Apache Tika gezeigt.
